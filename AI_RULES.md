# AI Project Rules

Este projeto usa Base44 + Vite + React.

## Regras obrigatórias
- Nunca modificar vite.config.*
- Nunca modificar src/api/base44Client.*
- Nunca alterar .env*
- Nunca modificar package.json
- Nunca duplicar imports
- Preservar alias @/
- Alterar apenas arquivos explicitamente solicitados
- Sempre mostrar diff antes de aplicar alterações
- Não reformatar arquivos inteiros
